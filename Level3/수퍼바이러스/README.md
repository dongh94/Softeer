2. 1 000 000 007 으로 나누어 나머지를 값으로 가질만큼 큰 값일 때

큰 수 모두에게 나누어주면서 계산해야 한다.

```python

def f(i, n):
    if n == 1:
        return i % 1000000007

    if n % 2 == 0:
        a = f(i, n // 2) % 1000000007
        return a  * a % 1000000007

    if n % 2 == 1:
        a = f(i, n // 2) % 1000000007
        return a * a * i % 1000000007

```
